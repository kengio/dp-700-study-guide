## Use Cases

- **Bookmark-and-scan** — keep this page open in a second tab while reading a topic file, and jump to the primary source when a fact needs more depth than the cheat sheets provide
- **Post-exam** — after passing, the same Fabric doc hubs stay relevant for real project work, not just certification
- **Link-rot check** — if a link 404s, it's a Microsoft doc reorganization; search the doc title on learn.microsoft.com and open a PR with the fix

## Exam Tips

> [!tip] Exam Tips
>
> - Microsoft Learn access during the exam is limited to learn.microsoft.com — the URLs in this page are all on that domain, so bookmarking a few of the densest ones (T-SQL surface area, KQL query overview, Activator limitations) is a legitimate in-exam lookup strategy for a genuinely unfamiliar syntax question
> - Don't rely on in-exam lookup as a primary strategy — the timer runs the whole time you're browsing, and most questions test applied judgment, not something you can find verbatim
