# Appendix

Reference materials for DP-700 exam preparation and troubleshooting — a glossary, the guide's full-size comparison tables in one place, and a consolidated error-message reference.

## Available Resources

| Resource | Description |
| :--- | :--- |
| [Glossary](./glossary.md) | ~65 Fabric/DP-700 terms A-Z, sourced from the guide's own topic files |
| [Comparison Tables](./comparison-tables.md) | The guide's full-size decision-matrix and comparison tables, collected in one place |
| [Error Messages](./error-messages.md) | Consolidated error reference from section 10 — one table per surface |

## How to Use

- **During study** — look up an unfamiliar term in the glossary as you read a topic file
- **Before the exam** — scan the comparison tables for a last pass on "choose between X/Y/Z" scenarios
- **While practicing** — reference the error-messages page when a practice question quotes an error string or code
- **On exam day** — all three pages are dense, table-heavy, and meant for fast lookup, not first-time learning; pair with the guide's other exam-morning resources once they land
